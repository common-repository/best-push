== Description ==

Send web push notifications without leaving wordpress panel with [best push](https://bestpush.io). 
moderately priced for all business levels.

**best push** offers you a Web Push Notifications plugin for your website. We support Chrome, Firefox, Opera, Microsoft Edge, 
on both desktop and mobile devices.

Now re-engage with customers with targeted push notifications. 
Within a few minutes, you can install and go live with Push Notifications on your website.

Once visitor subscribers to your push notifications, you can update visitors about your new post. 
The visitor will receive your push notifications even when they are not on your website.

# **Features:**

**Https and Http support:** **best push** Works on both HTTP and HTTPS sites.

**Analytics:** **best push** shows the number of your visitors and subscription per day. **best push** collects delivery and click count for every notification.

**Rich Elements:** put action buttons and image in your push notifications with **best push**. Also the Analysis data of buttons is accessible.

**Fast Setup:** Fast and easy to set up and no coding or API configuration required. All you have to do is installing the plugin and activating. The rest is on us.

**Increased Engagement:** Inform readers any time: when reading other websites or while working in other apps, with the browser closed. 
All missed notifications will be displayed after waking up even if computer is not active.

**Native:** Your readers won’t be bothered and don’t have to install any additional apps or plugins. They just accept receiving notifications from your website.

**Granular filtering:** Control the type of content to send out notifications. Filter by custom post type and by category for standard posts.

**Statistics:** The plugin provides essential usage and engagement data available directly in your WordPress dashboard so you can create the best experience for your subscribers.

== Installation ==

FROM YOUR WORDPRESS DASHBOARD

- Visit **Plugins** > **Add New**
- Search for **best-push**
- Install & activate **best-push** from your Plugins page.
- Now you can find best-push under the left menu.
